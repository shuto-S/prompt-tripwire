# PromptTripwire Issue #43 source-preview narration

Codex asks when it knows it is uncertain. PromptTripwire detects when
reasonable Codex runs silently disagree—and turns the human answer into an
execution contract.

PromptTripwire runs three independent read-only planning probes. Here, all
three probes are valid. This decision comes from both observed divergence and
deterministic policy, and the support counts show the split without exposing
raw probe IDs in the primary view.

The human selects local implementation. That answer creates an immutable
execution contract. It states what Codex may change, what must pass, and what
remains blocked. Approval is still a separate human action.

This source preview demonstrates version zero point one point twelve's
judge-facing presentation. It uses a deterministic fixture and is not a live
Codex inspect or execution recording.
